Quick start (Modded HDD builds)

Basic steps to get playing with the least effort.

Power on

  • Connect power + video cable
  • Insert FMCB memory card (if provided)
  • Turn on the console

Load games

  • Open your loader (e.g., OPL)
  • Browse the game list
  • Select title and launch

If you only see a few games, check the “Troubleshooting” section.

Troubleshooting (common)

Most issues are settings, input selection, or TV/cable related.

Games missing

  • Confirm you’re in the correct device/menu
  • Use page navigation in the loader
  • Confirm the HDD is detected

No signal / black screen

  • Switch TV input source
  • Try another HDMI port / adapter
  • Reseat cables

Freezing / loading issues

  • Some games need specific modes/settings
  • Try another title to compare
  • Message support with the exact game name
When contacting support, include: your TV model (if relevant), what cable/adapter you’re using, and a photo/video of the screen if possible.

Picture quality on modern TVs

Why PS2 can look rough on 4K TVs, and what actually improves it.

Why it looks worse on 4K OLED

  • PS2 outputs low-resolution analogue video
  • Modern TVs stretch it across a large panel
  • TV scalers often aren’t tuned for retro sources

What improves it most

  • Quality upscaler (best)
  • Good HDMI solution (varies by TV)
  • Correct TV settings (sharpness/processing)

Message your TV model and connection type for a realistic recommendation.
