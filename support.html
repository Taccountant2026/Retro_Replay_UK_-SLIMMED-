Quick fixes (common)

How do I exit a game back to the menu?

Try: L1 + R1 + L2 + R2 + Start + Select together (depends on game/settings).

No display / black screen?

Check video cable/adapter, try another TV input, and confirm the console boots without the kit.

Game not launching / freezing?

Try a different compatibility mode and confirm the drive/cables are seated firmly.
