What is FMCB + PS2 FAT modding?
Free McBoot (FMCB) is a memory card setup that boots PS2 homebrew tools. On PS2 FAT consoles, we can pair FMCB with a network adapter + HDD so games load from the drive (no discs). Built for simplicity: power on → open game list → play.
Modern TV note: PS2 outputs low-resolution video. On large 4K TVs, a good HDMI solution or upscaler may be recommended. See TV/picture help

Important clarity (no surprises)

PS2 video is low-resolution by modern standards. On large 4K TVs, image quality depends heavily on your HDMI adapter/upscaler. If you want the “YouTube look”, you typically need a good HDMI solution or upscaler.
